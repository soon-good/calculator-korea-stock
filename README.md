>

- 현재가가 직전 분기/연도별 영업이익/당기순이익 기준 몇배인지...

  - 생각해봤는데, 그런거 구현 안함. 시간 없음.

  - 조금 오래 걸리거나, 전문가적인 요소가 있다면 과감히 패스. 나는 주식쟁이가 아니다.
